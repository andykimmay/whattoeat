# 🍽️ WhatToEat

A simple meal decision app for two. Answer a few quick questions — weather, cuisine, eating in or out — and get a randomized meal suggestion from your personal list.

## Features

- **Smart filters**: Weather, cuisine type, dine in/out
- **Random picker**: Can't decide? Let the app choose
- **Manage meals**: Add, edit, delete your own meal options
- **Works offline**: Data saved in localStorage
- **Mobile-first**: Designed for phones, works everywhere

## Getting Started

```bash
npm install
npm run dev
```

## Deploy

This is a static site — deploy anywhere:

```bash
npm run build
```

Then upload the `dist/` folder to Vercel, Netlify, GitHub Pages, etc.

## Tech

- React + Vite
- No backend needed
- localStorage for persistence
